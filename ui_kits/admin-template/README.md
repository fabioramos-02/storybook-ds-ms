# UI Kit · Admin Template

Telas administrativas governamentais construídas com o DS-MS. Mimetiza o "Template - Login Administrativo", "Template - Página Interna" e "Template - Página de Erro" listados no menu Templates do Figma.

## Telas
- **Login Administrativo** — formulário centralizado, brand lockup, link "Esqueci minha senha"
- **Dashboard / Página Interna** — header, sidebar de seções, data table de cidadãos
- **Página de Erro 404** — full-screen com CTA de retorno

Navegação via state — clique nos botões/links para mudar de tela.

## Componentes
- `LoginScreen.jsx`
- `DashboardScreen.jsx` (header + sidebar + data table + tags de status)
- `ErrorScreen.jsx`
- `AdminSidebar.jsx`
- `DataTable.jsx`
